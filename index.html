Rail fence cipher

Rail fence pattern

A message can be reordered by writing it in a zigzag down and up across several rails, then reading each rail off left to right. To reverse it, place the letters back into the same zigzag shape and read them off in zigzag order—but only the correct number of rails will produce something readable. Vigenère cipher

Tabula recta

Used when you're given a keyword instead of a fixed shift—line the keyword up underneath the message, repeating it for the full length. For each letter: find the row labelled with that letter's keyword letter, then look along it to the column labelled with the plaintext letter—the cell where they meet is the ciphertext letter. (Decrypting works the same way in reverse: find the ciphertext letter in that row, then read off its column heading.)
